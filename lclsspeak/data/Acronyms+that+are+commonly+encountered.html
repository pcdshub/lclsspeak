Acronyms that are commonly encountered

Skip to end of metadata
  • Created by Mark Hunter, last modified on Sep 23, 2021
Go to start of metadata


Accounts

  • OIP = Operations Improvement Project
  • XIP = X-ray Improvement Project
  • AIP = Accelerator Improvement Project
  • M&S = Materials and Supplies


Endstations and Instruments

  • CXI = Coherent X-ray Imaging (hutch 5 but also a technique for bioimaging)
  • MFX = Macromolecular femtosecond crystallography
  • XCS = X-ray (photon) correlation spectroscopy
  • LJC = Liquid jet chamber.  Is deployed at either XCS or XPP and is a He enclosure
  • TMO = Time-resolved atomic and Molecular Imaging = NEH1.1
  • TXI = Tender X-ray Instrument = NEH1.2
  • L2SI = LCLS-II Scientific Instruments
  • chem-RIXS = chemistry Resonant Inelastic X-ray Scattering = NEH2.2
  • LAMP = ???
  • SC1 = Sample Chamber 1, main 1 um focus sample chamber in CXI
  • SC2 or SC01 = Sample Chamber 2, the 0.1 um focus sample chamber in CXI, commissioned second but is first along the beam path
  • SC3 = Sample Chamber 3, the refocusing chamber downstream of SC1; neither on same beam path as SC2
  • SSC = Sucky Sample Chamber, typically used for PCS or IH experiments

Techniques

  • SFX = Serial femtosecond crystallography
  • SMX = Serial macromolecular crystallography
  • SPI = Single particle imaging
  • DMA = Differential Mobility Analysis
  • DLS = Dynamic Light Scattering
  • MISC = Mix and Inject Serial Crystallography
  • SAXS = Small Angle X-ray Scattering
  • WAXS = Wide Angle X-ray Scattering
  • FXS = Fluctuation X-ray Scattering (also called fSAXS for fluctuation SAXS)
  • RIXS = Resonant Inelastic X-ray Scattering

Nozzles and injectors

  • GDVN = Gas dynamic virtual nozzle.  Common nozzle for aqueous-phase biosample delivery
  • DFFN = Double flow focus nozzle. Similar to GDVN but uses hydrodynamic focusing of a (outer, concentric) higher flow sheath liquid and low flow inner sample.
  • LCP = Lipidic Cubic Phase
  • LSP = Lipidic sponge phase
  • MESH = Mark Elaine Steven Hunter or Microfluidic Electrokinetic Sample Holder, originally written as nanoflow electrospinning injector
  • coMESH = concentric MESH

Groups/Centers/Facilities

  • SMB = Structural and Molecular Biology (Division at SSRL)
  • S2C2 = Stanford SLAC Cryo-EM Center
  • SED = Sample Environment and Delivery
  • CFEL = Center for Free Electron Laser Science
  • DESY = Deutsches Elektronen SYnchrotron
  • BioXFEL is a National Science Foundation center focused on XFEL studies

Other

  • IH = In house experiments (usually start with an X in the schedule)
  • PRP = Peer Review Panel
  • LDRD = Lab Directed Research and Development
  • SPL - Sample Prep Labs
  • UAA - User Access Agreement
  • ASC = Arrillaga Science Center
  • ICL = Injector Characterization Lab
  • RTRDF = Room Temperature Radiation Damage Free
  • PCS = Protein Crystal Screening
  • HPC = High Performance Computing
